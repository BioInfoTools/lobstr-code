1. chrom: chromosome number
2. start: start coordinate of the STR in hg19
3. end: end coordinate of the STR in hg19
4. ystr: YSTR marker name
5. hgdp: HGDP sample identifier
6. sample: SGDP sample identifier
7. allele.hgdp: Y-STR genotype reported in the HGDP capillary data
8. allele.sgdp: Y-STR genotype reported by lobSTR
9. coverage: number of reads used to make the lobSTR call
10. score: lobSTR quality score
