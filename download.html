New in version 2.0.8 (beta)

This version should have big alignment speedups compared to previous versions, up to 5-10x faster than before. This version also fixes a minor bug fix for an off-by-one error in how allele sequences are reported in the REF and ALT fields of the VCF file.

To see a list of changes and download previous lobSTR versions, see the changelog page.

lobSTR source code on Github

*Note*: the github repository is currently under construction as we get ready to release lobSTR v3 with some major changes to our STR calling algorithm. See the README there for more details.
The lobSTR source code is available on github (Note, we have moved over to github from sourceforge). To get the latest version, you can pull the code from the current git repo:

git clone https://github.com/mgymrek/lobstr-code.git lobstr-code

Then follow the installation instructions for building from source.

Contact mgymrek@mit.edu if you are interested in being involved with lobSTR development.

lobSTR index

The index for v2.0.3+ has been updated to expand a much broader range of STRs obtained by filtering Tandem Repeat Finder results using empirical score thresholds. Contains loci for periods 1 through 6 as well as markers obtained from CODIS, Marshfield and Y-STR datasets. More tools for how to compare markers at these annotated loci to datasets generated by other technologies is coming soon. Resource bundles contain a table of STR loci, the lobSTR index, and the file needed for the --strinfo option for the allelotyping step.
  • hg19 resource bundle

lobSTR v3. index

Starting with lobSTR v3.0.0+, we have changed our index structure. lobSTR v3 has not yet been released, but is currently available on github. If you use that version, you will need this new index. There is now one unified index, rather than files for each repeat motif. This index is not compatible with previous lobSTR versions. If you try to use the old index with v3.0.0+, you will receive an error message pointing you to where you can download the new index.
  • hg19 resource bundle for lobSTR v3
